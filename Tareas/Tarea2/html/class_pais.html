Descripción detallada

Gestiona información y operaciones relativas a un país.

La clase Pais ofrece funcionalidades para establecer y obtener atributos de un país como el nombre, población e identificador. Además, proporciona un método para generar un PIB aleatorio y sobrecarga el operador de igualdad para comparar países según la primalidad de sus identificadores.
