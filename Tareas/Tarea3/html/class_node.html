Detailed Description

Clase para representar un nodo en una lista enlazada de contactos.

Cada nodo contiene un puntero a un objeto Contacto, que almacena la información del contacto, y un puntero al siguiente nodo en la lista, formando así la estructura de la lista enlazada.
