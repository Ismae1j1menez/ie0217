Detailed Description

Clase que implementa una lista enlazada para gestionar contactos.

Esta clase utiliza nodos (Node) para almacenar y gestionar una lista enlazada de contactos. Proporciona métodos para agregar, eliminar, ordenar y mostrar contactos almacenados.
